What I have learnt so far using the HTML tags

Aside from the basic HTML structure tags i.e the html, head, title and body, I have learnt of the :

  1. header Tags which includes h1, h2, h3, h4, h5 and h6 in decreasing order with respect to the font size of their display
  2. p tags which is used to format paragraphs
  3. div tags which is used as a container to group several elements together
  4. em tags which is used to emphasize text
  5. i tag used to make text italic
  6. b tag used to make text bold
  7. strong tag also used to make text bold
  8. a tags called anchor tags used to create a link
  9. img tag used to add an image
  10. sub tag used to create a subscript
  11. sup tag used to create a superscript
  12. ul tag used to create unordered lists
  13. ol tag used to create ordered lists
  14. li tag used within the ul tags or ol tags to enclose the individual list items
